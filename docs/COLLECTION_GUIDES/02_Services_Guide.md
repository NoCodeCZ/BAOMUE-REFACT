# Services Collection Guide

## What is the Services Collection?

The **Services** collection manages all the services you offer. Each service can be categorized, linked to locations, and displayed throughout your website.

## Common Tasks

### Adding a New Service

1. Go to **Services** in the left sidebar
2. Click **Create Item** (or the "+" button)
3. Fill in the required fields (see Field Guide below)
4. Upload a hero image
5. Set **Status** to "Draft" while editing
6. Click **Save**
7. Change **Status** to "Published" when ready
8. Click **Save** again

### Editing an Existing Service

1. Go to **Services**
2. Find the service you want to edit (use search if needed)
3. Click on the service name
4. Make your changes
5. Click **Save**

### Organizing Services by Category

1. First, ensure you have **Service Categories** set up
2. When creating/editing a service, select a **Category**
3. Services will be grouped by category on your website

## Field-by-Field Guide

### Service Name
- **What it is:** The name of your service
- **Where it appears:** Service listings, detail pages, menus
- **Best practice:** Be clear and specific
- **Examples:** 
  - "Teeth Whitening"
  - "Dental Implants"
  - "Root Canal Treatment"

### URL Slug
- **What it is:** The URL part for the service detail page
- **Format:** Lowercase, use hyphens, no spaces
- **Examples:**
  - "teeth-whitening" → yoursite.com/services/teeth-whitening
  - "dental-implants" → yoursite.com/services/dental-implants
- **Tip:** Leave blank to auto-generate from service name

### Service Category
- **What it is:** The category this service belongs to
- **Purpose:** Organizes services on your website
- **How to use:** Select from dropdown (create categories first if needed)
- **Examples:** "General Dentistry", "Cosmetic Dentistry", "Orthodontics"

### Status
- **Draft:** Service not visible on website
- **Published:** Service is live and visible
- **Archived:** Service is hidden but kept for reference

### Short Description
- **What it is:** Brief summary (1-2 sentences)
- **Where it appears:** Service listings, cards, previews
- **Length:** Keep it concise (50-100 words)
- **Purpose:** Quick overview to grab attention
- **Example:** "Professional teeth whitening using advanced technology. Achieve a brighter smile in just one visit."

### Full Description
- **What it is:** Detailed information about the service
- **Where it appears:** Service detail page
- **Format:** Rich text (can use formatting, lists, headings)
- **Content ideas:**
  - What the service includes
  - Who it's for
  - Process/steps
  - Benefits
  - Aftercare information

### Duration
- **What it is:** How long the service typically takes
- **Format:** Free text
- **Examples:**
  - "30-45 minutes"
  - "1-2 hours"
  - "Multiple visits over 6 months"
- **Tip:** Be realistic about time expectations

### Price From
- **What it is:** Starting price for the service
- **Format:** Free text (include currency)
- **Examples:**
  - "฿2,500"
  - "Starting at $99"
  - "From £150"
- **Tip:** Leave empty if pricing varies significantly or requires consultation

### Hero Image
- **What it is:** Main image for the service
- **Where it appears:** Top of service detail page
- **Recommended size:** 1200x600px or larger
- **Format:** JPG or PNG
- **Tip:** Use high-quality, professional images

### SEO Title
- **What it is:** Title for search engines
- **Length:** 50-60 characters
- **Tip:** Include service name and location if relevant
- **Example:** "Teeth Whitening Bangkok | Professional Treatment"

### SEO Description
- **What it is:** Description for search results
- **Length:** 150-160 characters
- **Tip:** Write a compelling summary
- **Example:** "Professional teeth whitening in Bangkok. Safe, effective treatment using advanced technology. Book your appointment today!"

## Best Practices

### Writing Service Descriptions

1. **Start with benefits** - What will the customer gain?
2. **Be specific** - Avoid vague language
3. **Use clear language** - Avoid too much technical jargon
4. **Include process** - Explain what happens during the service
5. **Add trust signals** - Mention expertise, technology, guarantees

### Organizing Services

1. **Use categories** - Group related services
2. **Consistent naming** - Use similar naming patterns
3. **Complete information** - Fill all relevant fields
4. **Good images** - Use professional, clear images
5. **Regular updates** - Keep information current

### SEO Tips

1. **Use keywords naturally** - Include relevant search terms
2. **Unique descriptions** - Don't copy-paste between services
3. **Local keywords** - Include location if relevant (e.g., "Bangkok")
4. **Complete SEO fields** - Fill title and description
5. **Update regularly** - Fresh content helps SEO

## Examples

### Example: Creating a "Teeth Whitening" Service

1. Go to **Services** → Create Item
2. **Service Name:** "Teeth Whitening"
3. **Slug:** "teeth-whitening" (auto-generates)
4. **Category:** Select "Cosmetic Dentistry"
5. **Status:** Draft
6. **Short Description:** "Professional in-office teeth whitening treatment. Achieve a brighter, whiter smile in just one 60-minute session using advanced LED technology."
7. **Full Description:** 
   - Heading: "What is Teeth Whitening?"
   - Paragraph explaining the treatment
   - Heading: "Who is it for?"
   - List of candidates
   - Heading: "What to Expect"
   - Process steps
8. **Duration:** "60-90 minutes"
9. **Price From:** "฿3,500"
10. **Hero Image:** Upload professional image
11. **SEO Title:** "Teeth Whitening Bangkok | Professional Treatment | Tooth Box"
12. **SEO Description:** "Professional teeth whitening in Bangkok. Safe, effective treatment. Book your appointment today!"
13. Save as Draft
14. Review
15. Change to Published
16. Save

### Example: Editing Service Information

1. Go to **Services**
2. Find "Teeth Whitening"
3. Click to edit
4. Update price if changed
5. Add new information to description
6. Update image if you have a better one
7. Save

## Troubleshooting

**Q: My service isn't showing on the website**
- Check status is "Published"
- Verify category is set (if filtering by category)
- Make sure you clicked Save

**Q: I can't select a category**
- Go to **Service Categories** and create categories first
- Then return to Services and select the category

**Q: The slug is wrong**
- Edit the service
- Change the slug field
- Make sure it's unique (no other service uses it)
- Save

**Q: Images aren't showing**
- Go to **Files** and verify image uploaded
- Re-select the image in the service
- Check image format (JPG, PNG)
