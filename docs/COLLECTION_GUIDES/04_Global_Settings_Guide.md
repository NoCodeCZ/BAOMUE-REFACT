# Global Settings Guide

## What is Global Settings?

**Global Settings** is a special collection that contains site-wide settings affecting your entire website. Unlike other collections, there's only **one** Global Settings record - it's a "singleton" collection.

## Common Tasks

### Editing Site Information

1. Go to **Global Settings** in the left sidebar
2. Click to open (there's only one record)
3. Edit any fields you need
4. Click **Save**

### Changing Your Site Name

1. Open **Global Settings**
2. Find **Site Name** field
3. Enter your new site name
4. Click **Save**
5. Site name updates across the website

### Updating Your Logo

1. Open **Global Settings**
2. Find **Logo** field
3. Click the file selector
4. Choose existing logo or upload new one
5. Select the logo
6. Click **Save**

## Field-by-Field Guide

### Site Name
- **What it is:** Your website's name
- **Where it appears:** 
  - Header/navigation
  - Footer
  - Browser tab
  - Search results
- **Best practice:** Keep it concise and clear
- **Examples:**
  - "Tooth Box Dental Clinic"
  - "ABC Dental Services"
  - "Smile Care Dentistry"

### Site Description
- **What it is:** Brief description of your website/business
- **Where it appears:**
  - Search engine results
  - Social media sharing
  - Meta descriptions
- **Length:** 150-160 characters recommended
- **Best practice:** Write a compelling summary
- **Example:** "Professional dental services in Bangkok. Expert care, modern technology, comfortable environment. Book your appointment today!"

### Logo
- **What it is:** Your website logo
- **Where it appears:** Header/navigation bar
- **Recommended format:** PNG with transparent background
- **Recommended size:** At least 200px wide
- **File size:** Keep under 500KB for fast loading
- **Tip:** Use high-resolution logo for crisp display

### Favicon
- **What it is:** Small icon shown in browser tabs
- **Where it appears:** Browser tab, bookmarks
- **Format:** PNG or ICO
- **Size:** 32x32px or 64x64px (square)
- **Tip:** Should be simple and recognizable at small size

## Best Practices

### Site Name
- Keep it short and memorable
- Use your business name
- Be consistent across all platforms
- Don't change frequently (affects branding)

### Site Description
- Write for humans, not just search engines
- Include key information (location, services)
- Make it compelling
- Update if your business focus changes

### Logo
- Use high-quality image
- Ensure it works on light and dark backgrounds
- Test at different sizes
- Keep file size reasonable

### Favicon
- Use simplified version of logo
- Ensure it's recognizable at 16x16px
- Test in different browsers
- Update if rebranding

## Important Notes

### Singleton Collection
- There's only **one** Global Settings record
- You don't create new ones
- Just edit the existing one
- Changes affect the entire website

### Impact of Changes
- Changes to Global Settings affect the **entire website**
- Site name appears in header, footer, browser tabs
- Logo appears in navigation
- Description affects SEO and social sharing
- **Be careful** when making changes

### When to Update
- **Site Name:** When rebranding (rare)
- **Site Description:** When business focus changes
- **Logo:** When updating branding
- **Favicon:** When updating branding or logo

## Troubleshooting

**Q: I can't find Global Settings**
- Look in the left sidebar
- It may be under "Settings" or "Configuration"
- Contact administrator if not visible

**Q: My logo isn't showing**
- Check file uploaded in **Files**
- Re-select logo in Global Settings
- Verify file format (PNG recommended)
- Check file isn't corrupted

**Q: Logo looks blurry**
- Upload higher resolution version
- Ensure logo is at least 200px wide
- Use PNG format for best quality

**Q: Favicon isn't updating**
- Clear browser cache (Ctrl+F5 or Cmd+Shift+R)
- Wait a few minutes (browsers cache favicons)
- Try different browser
- Verify favicon file is correct format and size

**Q: Changes aren't appearing**
- Make sure you clicked **Save**
- Clear browser cache
- Check if changes are in Draft status (shouldn't be for Global Settings)
